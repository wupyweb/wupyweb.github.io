这两个是必须的，到网上下载安装就好了，也没什么难度，这里就不赘述。

1, 新安装好git后，我们需要进行一下设置（必须）

2，创建SSH key

创建过程中一路按回车即可，然后到保存秘钥的文件夹下，可以发现两个文件id_rsa和id_rsa.pub，分别为私钥和公钥。

在github的个人账户的setting的选项中添加ssh key，复制公钥的内容到key。
